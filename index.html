I ENJOY CREATING THINGS THAT SOLVE EVERY-DAY PROBLEMS

DiffRent

DiffRent is an inclusive, lightweight & easy-to-use platform for landlords and tenants to communicate. Whether you are a landlord with multiple apartments or a tenant that needs some work done, our ticketing system is easy to use for any issues that might arise in an apartment. Users can schedule appointments for site-visits & create and store messages within the app.
DiffRent was built using Javascript, React, Node.js, Express, PostgreSQL, HTML & CSS.

HikeMe!

HikeMe! helps outdoor enthusiasts find a hiking trail nearest them. This mobile friendly app gives you the ability to filter a list of hiking trails by distance & difficulty. Users of HikeMe! will get the name, image, rating, location & description of a trail that interests them so they can get out and surround themselves with nature in no-time.
HikeMe! was built by developers that love the outdoors and using technologies such as React, HTML, CSS and the Hiking Project API

Weather App

Wallet

Wallet is a personalized financial mobile application that encourages better spending and budget tracking. It connects to all of your financial accounts and gives you tools like data visualization to help track your spending, manage your budget and reach your financial goals. We’re also committed to keeping your data secure. With multiple safety measures like secure encryption and multi-factor authentication, we keep your information protected.
Wallet is built with Javascript, React Native, Redux, Node.js, Express, PostgreSQL, Charts.js, NativeBase & Plaid API
